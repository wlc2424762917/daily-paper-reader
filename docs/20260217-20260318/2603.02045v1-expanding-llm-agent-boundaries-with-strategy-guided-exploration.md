## 摘要
强化学习 (RL) 在将大语言模型 (LLM) 训练为执行计算机使用、工具调用和编程等任务的智能体方面取得了显著成功。然而，探索仍然是 LLM 智能体强化学习中的核心挑战，特别是在具有复杂观测和稀疏结果奖励的语言-动作空间中运行时。在这项工作中，我们通过利用 LLM 在语言中对环境进行规划和推理的能力，将探索从低级动作转向高级语言策略，从而解决了 LLM 智能体的探索问题。因此，我们提出了策略引导探索 (Strategy-Guided Exploration, SGE)，它首先生成描述如何朝着目标取得进展的简洁自然语言策略，然后根据该策略生成环境动作。通过在策略空间而非动作空间进行探索，SGE 引导了针对不同环境结果的结构化且多样化的探索。为了在强化学习期间增加策略多样性，SGE 引入了混合温度采样（并行探索多种策略）以及策略反思过程（基于环境中先前策略的结果来生成策略）。在 UI 交互、工具调用、编程和具身智能体环境中，SGE 一致优于专注于探索的强化学习基线，提高了学习效率和最终性能。我们展示了 SGE 使智能体能够学会解决对于基础模型而言过于困难的任务。

## Abstract
Reinforcement learning (RL) has demonstrated notable success in post-training large language models (LLMs) as agents for tasks such as computer use, tool calling, and coding. However, exploration remains a central challenge in RL for LLM agents, especially as they operate in language-action spaces with complex observations and sparse outcome rewards. In this work, we address exploration for LLM agents by leveraging the ability of LLMs to plan and reason in language about the environment to shift exploration from low-level actions to higher-level language strategies. We thus propose Strategy-Guided Exploration (SGE), which first generates a concise natural-language strategy that describes what to do to make progress toward the goal, and then generates environment actions conditioned on that strategy. By exploring in the space of strategies rather than the space of actions, SGE induces structured and diverse exploration that targets different environment outcomes. To increase strategy diversity during RL, SGE introduces mixed-temperature sampling, which explores diverse strategies in parallel, along with a strategy reflection process that grounds strategy generation on the outcomes of previous strategies in the environment. Across UI interaction, tool-calling, coding, and embodied agent environments, SGE consistently outperforms exploration-focused RL baselines, improving both learning efficiency and final performance. We show that SGE enables the agent to learn to solve tasks too difficult for the base model.

#### 1. 核心问题与整体含义（研究动机和背景）
在利用强化学习（RL）优化大语言模型（LLM）智能体时，**探索（Exploration）**是一个核心瓶颈。由于智能体运行在复杂的语言-动作空间（如代码、工具调用）中，且环境奖励通常极其稀疏（仅在任务完成时提供），传统的 RL 方法往往只能在模型已有的行为分布内进行微调，难以发现全新的解法。
*   **研究动机**：现有的 RL 训练（如 DeepSeek-R1 风格的训练）在非智能体推理任务中表现优异，但在智能体任务中，如果基础模型（Base Model）无法通过随机采样解决某个难题，RL 就无法获得学习信号。
*   **核心目标**：利用 LLM 的自然语言推理能力，将探索空间从“低级动作层级”提升到“高级策略层级”，从而突破基础模型的能力边界。

#### 2. 方法论：策略引导探索 (SGE)
SGE 的核心思想是让模型在执行动作前先生成一个**自然语言策略（Strategy）**，并围绕策略的多样性进行探索。

*   **核心流程**：
    1.  **策略提示（Strategy Prompting）**：要求模型在输出动作前，先生成一段简洁的自然语言描述，说明为了达成目标应该采取什么步骤。
    2.  **混合温度采样（Mixed-Temperature Sampling）**：在 RL 采样阶段，对“策略”部分的 Token 使用高温度（High Temperature），以诱导生成多样化的解题思路；对后续的“动作”部分使用低温度，以确保动作执行的精确性。
    3.  **策略反思（Strategy Reflection）**：
        *   **负向反思**：如果之前的尝试失败了，将失败的策略作为上下文输入，要求模型分析原因并生成一个截然不同的新策略。
        *   **正向反思**：利用成功的历史策略引导模型巩固并优化成功路径。
*   **算法框架**：基于 **GRPO**（Group Relative Policy Optimization）算法，通过组内相对奖励来估计优势函数，无需额外的 Critic 模型。

*   **客观性**：通过对比 `pass@k` 曲线，客观展示了 SGE 如何突破基础模型的性能天花板（即即使基础模型尝试 2048 次也解不出的题，SGE 训练后能解出）。实验设计覆盖了从视觉 UI 到纯代码逻辑的多种维度，具有较强的说服力。

#### 6. 主要结论与发现
*   **突破性能天花板**：SGE 能够解决基础模型在海量采样下（如 pass@2048）仍无法解决的难题，而传统 RL 基线往往受限于基础模型的能力上限。
*   **提升学习效率**：在所有测试环境中，SGE 的收敛速度和最终成功率均显著优于 EntropyAdv 和 RND 等探索基线。
*   **泛化能力**：在训练集之外的未见任务（Unseen Tasks）上，SGE 训练的模型表现出更强的迁移能力。
*   **规模效应**：SGE 的效果随模型规模增大而增强。在 600M 模型上效果有限，但在 4B 和 8B 模型上表现卓越，说明该方法依赖于模型本身的语言推理底座。

#### 7. 优点
*   **思路新颖**：将 RL 的探索从 Token 级别的随机性提升到了语义级别的策略多样性，符合 LLM 的特性。
*   **即插即用**：作为一种采样和提示策略，它可以与任何在线 RL 算法（如 PPO, GRPO）结合。
*   **实用性强**：通过混合温度解决了“探索多样性”与“执行准确性”之间的矛盾。

#### 8. 不足与局限
*   **推理开销**：由于每一步动作前都要生成一段策略描述，增加了训练和推理时的 Token 消耗及响应延迟。
*   **底座依赖**：如果基础模型本身的推理能力极差（如参数量过小），则无法生成有效的策略，导致 SGE 失效。
*   **长程任务挑战**：在步骤极多的任务中，存储和反思全量历史策略可能会导致上下文窗口过长，文中提到未来可能需要对策略进行摘要处理。
