## 摘要
在处理长程复杂任务（如多轮对话、游戏博弈、科学发现）时，记忆已成为基于大语言模型（LLM）的智能体的核心模块，它能够实现知识积累、迭代推理和自我进化。文献中已经提出了许多记忆方法。然而，这些方法尚未在相同的实验设置下得到系统且全面的比较。在本文中，我们首先从高层视角总结了一个整合了所有现有智能体记忆方法的统一框架。随后，我们在两个知名基准测试上对具有代表性的智能体记忆方法进行了广泛比较，并检验了所有方法的有效性，提供了对这些方法的深入分析。作为实验分析的副产品，我们还通过利用现有方法中的模块设计了一种新的记忆方法，其性能优于现有最先进的方法。最后，基于这些发现，我们提出了具有前景的未来研究方向。我们相信，深入理解现有方法的行为可以为未来的研究提供宝贵的新见解。

## Abstract
Memory emerges as the core module in the large language model (LLM)-based agents for long-horizon complex tasks (e.g., multi-turn dialogue, game playing, scientific discovery), where memory can enable knowledge accumulation, iterative reasoning and self-evolution. A number of memory methods have been proposed in the literature. However, these methods have not been systematically and comprehensively compared under the same experimental settings. In this paper, we first summarize a unified framework that incorporates all the existing agent memory methods from a high-level perspective. We then extensively compare representative agent memory methods on two well-known benchmarks and examine the effectiveness of all methods, providing a thorough analysis of those methods. As a byproduct of our experimental analysis, we also design a new memory method by exploiting modules in the existing methods, which outperforms the state-of-the-art methods. Finally, based on these findings, we offer promising future research opportunities. We believe that a deeper understanding of the behavior of existing methods can provide valuable new insights for future research.

### 1. 核心问题与研究动机
在长程复杂任务（如多轮对话、游戏、科学发现）中，记忆模块是 LLM 智能体实现知识积累、迭代推理和自我进化的核心。尽管目前已涌现出大量记忆方法，但研究界仍面临以下挑战：
*   **缺乏统一框架**：现有方法设计迥异，缺乏系统性的抽象和分类。
*   **缺乏公平对比**：各方法通常在不同的实验设置下报告结果，难以直接衡量优劣。
*   **组件作用不明**：很少有研究深入探讨记忆系统内部各组件（如提取、存储、检索）的具体贡献。

### 2. 方法论
论文提出了一个**统一的模块化框架**，将智能体记忆系统分解为四个核心阶段：
*   **信息提取 (Information Extraction)**：包括直接归档（原始消息）、基于摘要的提取（关键词/标签）和基于图的提取（实体/关系三元组）。
*   **记忆管理 (Memory Management)**：模拟人类记忆生命周期，包含连接（关联相关经验）、整合（抽象/总结）、转换（短长期迁移）、更新（修正/合并）和过滤（删除陈旧信息）。
*   **记忆存储 (Memory Storage)**：从组织维度分为扁平存储和分层存储；从表示维度分为向量存储和图/树状存储。
*   **信息检索 (Information Retrieval)**：涵盖词法检索（BM25）、向量检索（语义相似度）、结构化检索（图遍历）和 LLM 辅助检索。

**新方法设计**：基于实验分析，作者设计了一种结合树状组织与分层存储的新方法。它将记忆分为短、中、长期三层，利用语义相似度进行分段处理，并通过“热度”评分（频率+近时性）将重要片段晋升至长期记忆，实现了高性能与低成本的平衡。

### 3. 实验设计
*   **数据集/场景**：
    *   **LOCOMO**：基于人类对话的 QA 任务，包含单跳/多跳检索、时序推理等。
    *   **LONGMEMEVAL**：基于用户-AI 交互的长程记忆基准，平均对话长度达 11.5 万 token。
*   **对比方法**：选取了 10 种代表性方法，包括 A-MEM, MemoryBank, MemGPT, Mem0, Mem0g, MemoChat, Zep, MemTree, MemoryOS, MemOS。
*   **评估指标**：F1 分数和 BLEU-1（衡量 token 重叠度和词法忠实度）。
*   **分析维度**：Token 成本效率、上下文可扩展性（50%-200% 长度）、证据位置敏感性（早/中/晚）、LLM 底座依赖性。

### 4. 资源与算力
*   **硬件**：使用了 **8 台 NVIDIA A100 (80 GB) GPU**。
*   **模型底座**：主要使用 Qwen2.5-7B/72B-Instruct，辅助测试了 LLaMA3.1-8B 和 GPT-4o-mini。
*   **时长限制**：实验设定了 2 天的运行阈值，若方法在 2 天内无法完成推理则视为失败（如 Zep 在某些长任务中）。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了 10 种基准方法在两个大型数据集上的表现，并针对 4 个维度进行了深入的鲁棒性分析。
*   **充分性与公平性**：实验设计非常充分且客观。作者统一了 Embedding 模型（all-MiniLM-L6-v2）、检索数量（k=10）以及输出简化处理（Answer Simplification），确保了不同架构之间的横向对比是公平的。

### 6. 主要结论与发现
*   **架构优势**：树状和分层记忆结构（如 MemTree, MemOS）在处理复杂任务时显著优于扁平结构。
*   **信息完整性**：保留原始对话片段（Raw Messages）至关重要，过度依赖三元组提取会导致语义丢失。
*   **Token 成本**：记忆管理的粒度决定了成本。将多轮对话合并为“段落”处理能大幅降低 Token 消耗而不损失太多精度。
*   **鲁棒性挑战**：所有方法都表现出“近因偏差”（对近期信息的检索更准），且随着上下文规模扩大，噪声干扰会导致性能稳步下降。

### 7. 优点
*   **系统性强**：首次为 LLM 记忆方法建立了清晰的模块化分类学。
*   **实战导向**：不仅关注准确率，还深入分析了工业界关心的 Token 成本和长文本扩展性。
*   **启发性**：通过组合现有模块设计出的新方法直接达到了 SOTA 水平，证明了框架的指导价值。

### 8. 不足与局限
*   **模态局限**：目前主要集中在文本记忆，未涉及音频、图像等多模态信号的统一记忆管理。
*   **压缩瓶颈**：虽然讨论了管理策略，但对于如何在不丢失关键信息的前提下实现极高压缩比的记忆表示仍缺乏突破。
*   **动态交互**：基准测试多为静态历史记录，未能完全模拟真实世界中记忆随交互实时演化的动态过程。
