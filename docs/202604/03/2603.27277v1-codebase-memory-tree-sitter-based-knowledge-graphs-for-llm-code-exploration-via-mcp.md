## 摘要
大语言模型 (LLM) 编程智能体通常通过重复的文件读取和 grep 搜索来探索代码库，每次查询消耗数千个 token，且缺乏结构化理解。我们提出了 Codebase-Memory，这是一个开源系统，它通过模型上下文协议 (MCP) 构建持久的、基于 Tree-Sitter 的知识图谱。该系统通过包含并行工作池、调用图遍历、影响分析和社区发现的多阶段流水线，支持对 66 种语言的解析。在 31 个真实代码库的评估中，Codebase-Memory 实现了 83% 的回答质量（相比之下，文件探索智能体为 92%），但 token 消耗减少了 10 倍，工具调用次数减少了 2.1 倍。对于诸如核心节点检测 (hub detection) 和调用者排名 (caller ranking) 等图原生查询，它在 31 种语言中的 19 种上达到或超过了探索智能体的表现。

## Abstract
Large Language Model (LLM) coding agents typically explore codebases through repeated file-reading and grep-searching, consuming thousands of tokens per query without structural understanding. We present Codebase-Memory, an open-source system that constructs a persistent, Tree-Sitter-based knowledge graph via the Model Context Protocol (MCP), parsing 66 languages through a multi-phase pipeline with parallel worker pools, call-graph traversal, impact analysis, and community discovery. Evaluated across 31 real-world repositories, Codebase-Memory achieves 83% answer quality versus 92% for a file-exploration agent, at ten times fewer tokens and 2.1 times fewer tool calls. For graph-native queries such as hub detection and caller ranking, it matches or exceeds the explorer on 19 of 31 languages.

#### 1. 核心问题与整体含义（研究动机和背景）
当前 LLM 编程智能体（如 Claude Code、Cursor、Aider）在探索代码库时，主要依赖“文件读取”和“grep 搜索”等文本级操作。这种方式存在两个核心痛点：
*   **效率低下与成本高昂**：为了理解代码结构（如“修改此函数会影响什么？”），智能体需要进行数十次工具调用，消耗数十万个 Token，且容易在长上下文中丢失关键信息。
*   **结构理解缺失**：代码本质上是结构化的（调用图、依赖链），而文本搜索无法直接捕获传递性关系，导致智能体难以处理复杂的架构问题。

**Codebase-Memory** 旨在通过构建一个持久化的、基于知识图谱的检索层，将代码结构作为“一等公民”直接暴露给 LLM，从而实现高效、低成本的代码探索。

#### 2. 方法论：核心思想与技术细节
该系统采用 C 语言实现，作为一个单二进制文件运行，通过 **Model Context Protocol (MCP)** 与 LLM 交互。其核心流程分为三个阶段：

*   **解析阶段 (Parse)**：利用 **Tree-Sitter** 解析 66 种编程语言，提取函数、类、接口、调用点、导入关系等。针对 Go/C/C++，结合了类似 LSP 的类型解析以处理指针和方法接收者。
*   **构建阶段 (Build)**：

    *   **社区发现**：应用 **Louvain 算法** 对调用图进行模块化优化，自动识别功能社区。
    *   **增量同步**：使用 **XXH3 哈希** 监测文件变化，仅对受影响文件进行增量索引。
*   **服务阶段 (Serve)**：通过 MCP 暴露 14 个结构化查询工具（如 `trace_call_path`、`search_graph`、`detect_changes`），支持类 Cypher 的图查询语言。

#### 8. 不足与局限
