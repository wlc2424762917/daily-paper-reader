## 摘要
大语言模型（LLMs）日益依赖智能体能力（如迭代检索、工具使用和决策）来克服静态参数化知识的局限性。然而，现有的智能体框架将外部信息视为非结构化文本，未能利用现实世界数据中固有的拓扑依赖关系。为了弥补这一差距，我们引入了智能体图学习（Agentic Graph Learning, AGL），这是一种将图学习重新定义为拓扑感知导航与基于 LLM 推理的交替过程的新范式。具体而言，我们提出了 AgentGL，这是首个用于 AGL 的强化学习（RL）驱动框架。AgentGL 为 LLM 智能体配备了用于多尺度探索的图原生工具，通过搜索约束思维来规范工具使用以平衡准确性与效率，并采用图条件课程强化学习策略，在无需逐步监督的情况下稳定长程策略学习。在多种文本属性图（TAG）基准测试和多个 LLM 主干模型上，AgentGL 的表现显著优于强力的 GraphLLM 和 GraphRAG 基准模型，在节点分类和链路预测中分别实现了高达 17.5% 和 28.4% 的绝对提升。这些结果表明，AGL 是使 LLM 能够在复杂关系环境中自主导航和推理的一个极具前景的前沿方向。代码已在 https://github.com/sunyuanfu/AgentGL 公开。

## Abstract
Large Language Models (LLMs) increasingly rely on agentic capabilities-iterative retrieval, tool use, and decision-making-to overcome the limits of static, parametric knowledge. Yet existing agentic frameworks treat external information as unstructured text and fail to leverage the topological dependencies inherent in real-world data. To bridge this gap, we introduce Agentic Graph Learning (AGL), a paradigm that reframes graph learning as an interleaved process of topology-aware navigation and LLM-based inference. Specifically, we propose AgentGL, the first reinforcement learning (RL)-driven framework for AGL. AgentGL equips an LLM agent with graph-native tools for multi-scale exploration, regulates tool usage via search-constrained thinking to balance accuracy and efficiency, and employs a graph-conditioned curriculum RL strategy to stabilize long-horizon policy learning without step-wise supervision. Across diverse Text-Attributed Graph (TAG) benchmarks and multiple LLM backbones, AgentGL substantially outperforms strong GraphLLMs and GraphRAG baselines, achieving absolute improvements of up to 17.5% in node classification and 28.4% in link prediction. These results demonstrate that AGL is a promising frontier for enabling LLMs to autonomously navigate and reason over complex relational environments. The code is publicly available at https://github.com/sunyuanfu/AgentGL.

*   **研究动机**：
    *   **静态局限性**：现有的图大模型（GraphLLMs）通常在推理时提取一次性静态上下文，缺乏自适应探索能力。
    *   **结构缺失**：传统的检索增强生成（RAG）和智能体搜索主要基于词法相似性，无法利用图的结构信号。

*   **目标**：提出“智能体图学习”（Agentic Graph Learning, AGL）范式，让 LLM 能够像在互联网搜索一样，在图结构中自主导航、收集证据并迭代推理。

### 2. 方法论
AgentGL 是首个由强化学习（RL）驱动的 AGL 框架，其核心由三个部分组成：
*   **图原生搜索工具（GNS Tools）**：为 LLM 配备了四种多尺度探索工具：
    *   **1-hop/2-hop 邻域搜索**：基于拓扑连接和查询相关性检索局部证据。
    *   **结构显著性搜索（SS）**：利用 PageRank 等结构先验识别全局关键节点。
    *   **图稠密搜索（DENSE）**：通过语义向量相似度跨越拓扑孤岛检索相关节点。
*   **两阶段强化学习策略**：
    *   **阶段 1：策略引导（Bootstrapping）**：通过奖励建模（包含格式奖励、准确性奖励和工具覆盖率奖励），教会 LLM 熟练使用图搜索工具。
    *   **阶段 2：缓解搜索过度（MSO）**：引入“搜索约束思维”（Search-Constrained Thinking），通过回顾性终止触发器和认知密度正则化，惩罚冗余搜索，鼓励深度推理，实现效率与准确性的平衡。
*   **图条件课程学习（GCCL）**：
    *   利用图的内在属性（如节点分类中的同质性和度数、链路预测中的语义-标签一致性）自动评估样本难度。
    *   训练过程遵循“由易到难”的顺序，以稳定长程策略的收敛。
*   **算法实现**：支持 GRPO 和 REINFORCE++ 两种主流的无评论员（Critic-free）策略优化算法。

*   **性能领先**：AgentGL 在所有基准测试中均取得最优结果。相比基准模型，节点分类准确率提升高达 **17.5%**，链路预测提升高达 **28.4%**。

### 7. 优点
*   **范式创新**：首次将图学习定义为 LLM 智能体的自主导航与推理过程，统一了结构感知与语义推理。
*   **训练稳定**：引入课程学习（GCCL）解决了长程图搜索策略难以收敛的问题，无需昂贵的逐步轨迹标注。
*   **工具完备**：设计的 GNS 工具箱兼顾了局部与全局、拓扑与语义，覆盖了图探索的多个维度。

*   **模态限制**：目前仅支持文本属性图，尚未扩展到包含图像等多模态属性的图数据。

*   **图密度挑战**：在极稠密的图中，搜索空间会急剧膨胀，如何进一步优化搜索效率仍是潜在挑战。
