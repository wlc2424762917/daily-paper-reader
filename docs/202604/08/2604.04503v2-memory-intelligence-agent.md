## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 的推理能力与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储和检索成本增加等关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，该框架由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可以存储压缩后的历史搜索轨迹。规划者是一个参数化记忆智能体，能够为问题生成搜索计划。执行者是另一个在搜索计划指导下进行信息搜索和分析的智能体。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续演化，在不中断推理过程的情况下，随推理同步进行即时更新。此外，我们在参数化记忆和非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思和无监督判断机制，以提升在开放世界中的推理和自我演化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

### 1. 核心问题与整体含义（研究动机和背景）
*   **核心问题**：现有的深度研究智能体在处理长程、多轮交互任务时，其记忆系统存在四大瓶颈：
    1.  **注意力稀释**：长上下文记忆会导致模型注意力分散。
    2.  **噪声干扰**：无关或弱相关的历史信息会降低推理质量。
    3.  **存储与检索成本高昂**：随着任务增加，记忆库无限膨胀。
    4.  **缺乏过程性记忆**：现有系统多存储“事实结果”，而忽略了“如何获得结果”的搜索策略和失败教训。
*   **研究动机**：受人类大脑启发，DRA 需要一种既能存储显式经验（非参数化），又能将经验内化为推理能力（参数化）的记忆机制，从而实现“边做边学”的持续进化。

### 2. 方法论：核心思想与关键技术
MIA 采用了 **管理者-规划者-执行者 (Manager-Planner-Executor)** 的解耦架构：
*   **核心思想**：将记忆分为“非参数化显式记忆”（存储压缩后的工作流）和“参数化隐式记忆”（通过训练更新规划者的权重），并通过测试时学习（TTL）实现两者的双向转换。
*   **关键技术细节**：
    *   **记忆管理者 (Manager)**：负责将冗长的搜索轨迹压缩为结构化的“工作流摘要”，并基于语义相似度、价值奖励（成功率）和频率奖励进行混合检索。
    *   **规划者 (Planner)**：作为认知中枢，利用检索到的记忆生成分步搜索计划，并具备“反思-重规划”能力。
    *   **执行者 (Executor)**：负责具体的工具调用（如网页搜索、图像搜索）和信息分析。
    *   **交替强化学习 (Alternating RL)**：基于 GRPO 算法，第一阶段训练执行者遵循计划，第二阶段训练规划者吸收记忆并优化策略。
    *   **测试时学习 (TTL)**：在推理过程中同步进行参数更新，无需中断工作流即可实现即时进化。
    *   **无监督判断机制**：模拟学术评审过程（Reviewer-AC 机制），从逻辑一致性、信息可信度和结果有效性三个维度对轨迹进行自动评估，支持在无标注环境下的自我演化。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有 SOTA 模型的表现。例如，将 GPT-5.4 在 LiveVQA 上的表现提升了 9%。
*   **以小博大**：搭载 MIA 的 **7B 规模执行者** 平均性能提升 31%，甚至超过了未搭载该框架的 32B 规模模型（领先 18%）。
*   **持续进化**：通过 TTL，模型在多次迭代中表现出稳步增长的趋势（如在无监督设置下从 59.6 提升至 61.7）。
*   **记忆效率**：相比传统的长上下文记忆，MIA 的压缩工作流机制有效减少了噪声，提高了多轮工具调用的成功率。

### 7. 优点（亮点）
*   **架构创新**：通过“管理者-规划者-执行者”的解耦，解决了记忆存储与推理效率的矛盾。
*   **双向记忆转换**：实现了非参数化经验向参数化能力的转化，这在智能体领域具有前瞻性。
*   **无监督演化**：提出的“类同行评审”判断机制，为智能体在缺乏人类反馈的开放世界中生存提供了可行方案。
*   **通用性强**：实验证明该框架不仅适用于开源模型，也能有效增强闭源商业模型。

### 8. 不足与局限
*   **计算开销**：虽然推理更精准，但 TTL 过程涉及在线参数更新和多次 Rollout，可能会增加单次查询的延迟和算力成本。
*   **任务领域限制**：目前主要集中在“深度研究”（Deep Research）类任务，对于实时性要求极高或纯创意类的任务，该框架的适用性尚待验证。
*   **多智能体协同复杂性**：三个角色之间的通信和同步在极端复杂环境下可能存在稳定性风险。
