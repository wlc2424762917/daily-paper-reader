## 摘要
大语言模型（LLMs）越来越多地被部署在部分可观测环境下的长程任务中，在这些任务中，模型必须在执行动作的同时，跨多个步骤推断并追踪复杂的环境状态。这带来了两个挑战：部分可观测性要求对未观测到的世界属性保持不确定性，而漫长的交互历史导致上下文无限制增长，稀释了与任务相关的信息。解决这两个挑战的一个原则性方案是信念状态（belief state）：即给定过去观测和动作后，关于环境状态的后验分布，它能够紧凑地编码历史信息以供决策，且与回合长度无关。然而，在 LLM 智能体中，文本的开放性使得如何表示这种分布变得不明确。因此，我们提出了 Agent-BRACE（通过抽象和置信度估计实现的智能体信念状态表示），这是一种将 LLM 智能体解耦为信念状态模型和策略模型的方法，两者通过强化学习进行联合优化。信念状态模型生成信念分布的结构化近似：一组关于环境的原子级自然语言断言，每个断言都标注了从“确定”到“未知”的序数言语化确定性标签。策略模型以这种紧凑、结构化的近似信念而非完整历史为条件，学习在显式不确定性下选择动作。在长程、部分可观测的具身语言环境中，Agent-BRACE 实现了 +14.5% (Qwen2.5-3B-Instruct) 和 +5.3% (Qwen3-4B-Instruct) 的平均绝对提升，在优于强强化学习基线的同时，保持了与回合长度无关的近乎恒定的上下文窗口。进一步分析表明，随着证据的积累，学习到的信念在回合过程中变得越来越校准。

## Abstract
Large language models (LLMs) are increasingly deployed on long-horizon tasks in partially observable environments, where they must act while inferring and tracking a complex environment state over many steps. This leads to two challenges: partial observability requires maintaining uncertainty over unobserved world attributes, and long interaction history causes context to grow without bound, diluting task-relevant information. A principled solution to both challenges is a belief state: a posterior distribution over environment states given past observations and actions, which compactly encodes history for decision making regardless of episode length. In LLM agents, however, the open-ended nature of text makes it unclear how to represent such a distribution. Therefore, we introduce Agent-BRACE: Agent Belief state Representation via Abstraction and Confidence Estimation, a method that decouples an LLM agent into a belief state model and a policy model, jointly optimized via reinforcement learning. The belief state model produces a structured approximation of the belief distribution: a set of atomic natural language claims about the environment, each annotated with an ordinal verbalized certainty label ranging from certain to unknown. The policy model conditions on this compact, structured approximate belief rather than the full history, learning to select actions under explicit uncertainty. Across long-horizon, partially observable embodied language environments, Agent-BRACE achieves an average absolute improvement of +14.5% (Qwen2.5-3B-Instruct) and +5.3% (Qwen3-4B-Instruct), outperforming strong RL baselines while maintaining a near-constant context window independent of episode length. Further analysis shows that the learned belief becomes increasingly calibrated over the course of an episode as evidence accumulates.

### 1. 核心问题与研究背景
在长程（Long-Horizon）且部分可观测（Partially Observable）的任务中，大语言模型（LLM）智能体面临两大挑战：
*   **部分可观测性（Partial Observability）：** 智能体无法一次性获取环境全貌，必须在行动中推断隐藏状态，并对未观测到的属性保持不确定性。
*   **上下文膨胀（Context Bloat）：** 随着交互步数增加，历史记录无限增长，不仅增加计算开销，还会稀释关键信息，导致模型决策质量下降。

**研究动机：** 借鉴控制理论中的“信念状态（Belief State）”概念，即对环境状态的后验概率分布。论文旨在探索如何在 LLM 中以自然语言形式表示这种分布，从而实现历史信息的压缩与不确定性的显式建模。

### 2. 方法论：Agent-BRACE 框架
Agent-BRACE 的核心思想是将智能体解耦为两个协同工作的模型，并通过强化学习（RL）进行联合优化：

*   **信念状态模型（Belief State Model, BSM）：**
    *   **核心功能：** 接收当前观测和上一步的信念，更新并输出一组关于环境状态的“原子级断言”（Atomic Claims）。
    *   **言语化不确定性：** 每个断言都附带一个序数形式的置信度标签（如：确定 Certain、可能 Likely、不确定 Unlikely、未知 Unknown）。
    *   **状态更新：** $b_t = \text{BSM}(o_t, a_{t-1}, b_{t-1})$，其中 $b$ 是信念，$o$ 是观测，$a$ 是动作。
*   **策略模型（Policy Model, PM）：**
    *   **核心功能：** 仅根据 BSM 提供的紧凑信念状态（而非原始历史记录）来决定下一步动作。
    *   **决策逻辑：** $a_t = \text{PM}(b_t)$。
*   **联合强化学习优化：**
    *   使用 **PPO（Proximal Policy Optimization）** 算法。
    *   **奖励函数：** 包含任务成功奖励（针对 PM）和信念准确性/校准奖励（针对 BSM）。通过这种方式，BSM 学习提取对决策最有用的信息，而 PM 学习在显式不确定性下进行稳健决策。

### 6. 主要结论与发现
*   **性能显著提升：** 在 ALFWorld 和 ScienceWorld 上，Agent-BRACE 相比强 RL 基线分别实现了最高 **14.5%** 和 **5.3%** 的绝对性能提升。

*   **解耦的价值：** 明确区分“我相信什么”和“我该做什么”有助于模型在复杂逻辑推理中减少幻觉。

### 7. 优点与亮点
*   **架构优雅：** 将经典的信念状态理论与现代 LLM 的自然语言能力结合，实现了可解释性与效率的平衡。
*   **高效性：** 实现了与回合长度无关的近乎恒定的上下文开销，这对实际部署具有重要意义。
*   **可解释性：** 智能体的“内心世界”（信念）通过自然语言断言显式表达，人类可以直接观察到智能体在哪里产生了误解或不确定。

### 8. 不足与局限
*   **训练复杂性：** 联合优化两个模型（BSM 和 PM）比单一模型训练更复杂，对奖励函数的设计较为敏感。

*   **潜在的信念幻觉：** 虽然有校准奖励，但 BSM 仍可能生成与实际观测不符的错误断言，这种错误会直接误导 PM。
