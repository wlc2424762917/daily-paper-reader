## 摘要
基于大语言模型的多智能体系统（MAS）在处理复杂任务方面展现出了卓越的能力。然而，手动设计最优通信拓扑不仅耗费人力，且自动扩展方法往往会导致结构臃肿并产生冗余智能体，从而导致过度的 Token 消耗。为解决这一问题，我们提出了 AgentSlimming，这是一种用于图结构多智能体工作流的即插即用压缩框架。受神经网络剪枝和量化的启发，AgentSlimming 通过混合机制首先估计每个智能体的重要性得分，进而压缩工作流，随后移除冗余智能体或将其替换为低成本智能体。每项操作均通过基于基准锚定的接受规则进行验证，以防止性能崩溃。实验结果表明，AgentSlimming 在性能损失极小的情况下，可将平均 Token 成本降低高达 78.9%，有时甚至能提升准确率，在成本与质量之间实现了显著的帕累托最优权衡。我们的代码已公开于 https://github.com/CitrusYL/AgentSlimming。

## Abstract
Large Language Model-based Multi-Agent Systems (MAS) have demonstrated remarkable capabilities in complex tasks. However, manually designing optimal communication topologies is labor-intensive, while automated expansion methods often result in bloated structures with redundant agents, leading to excessive token consumption. To address this problem, we introduce \textbf{AgentSlimming}, a plug-and-play compression framework for graph-structured multi-agent workflows. Motivated by pruning and quantization in neural networks, AgentSlimming compresses workflows by first estimating the importance score of each agent with a hybrid mechanism, and then removes redundant agents or replaces them with low-cost ones, where each operation is validated using a baseline-anchored acceptance rule to prevent performance collapse. Experiments show that AgentSlimming reduces average token cost by up to 78.9\% with negligible performance degradation, and sometimes even improves accuracy, achieving a strong Pareto-optimal trade-off between cost and quality. \textit{Our code is publicly available at https://github.com/CitrusYL/AgentSlimming

### 1. 核心问题与整体含义（研究动机）
*   **研究背景**：当前 MAS 的设计正从手动构建转向自动化搜索（如 AFlow、ADAS）。虽然自动化方法能发现高性能的拓扑结构，但通常假设“资源无限制”，导致生成的系统包含大量冗余智能体和密集的通信路径。
*   **核心问题**：现有的自动化 MAS 结构臃肿，所有节点往往统一使用最昂贵的模型（如 GPT-4），导致 Token 消耗呈二次方增长，难以大规模扩展。
*   **研究目标**：开发一种即插即用的压缩框架，在保持（甚至提升）性能的前提下，大幅降低 MAS 的运行成本。

### 2. 方法论：核心思想与技术细节
AgentSlimming 借鉴了神经网络的压缩思想，提出了一个三阶段的流水线：
*   **混合重要性评估（Hybrid Importance Evaluation）**：这是决定“删谁”或“换谁”的关键。它综合了四种信号：
    1.  **拓扑信号**：度中心性（Degree Centrality）和介数中心性（Betweenness Centrality），衡量节点在图结构中的位置重要性。
    2.  **功能信号**：近似 Shapley 值。通过“留一法”（Leave-One-Out, LOO）估计移除某个节点对整体性能的影响。
    3.  **经济信号**：Delta-Cost，衡量移除该节点能节省多少美元。
    *   **排序融合**：使用倒数排名融合（RRF）将上述异构指标整合为一个鲁棒的重要性得分。
*   **三个核心组件**：
    1.  **AgentPruner（结构剪枝）**：按重要性得分从低到高，迭代移除冗余节点。若移除后性能下降超过阈值，则触发回滚。
    2.  **AgentQuant（语义量化）**：将非核心节点的昂贵模型（如 GPT-4.1-mini）替换为低成本模型（如 GPT-4.1-nano）。
    3.  **AgentTuner（自适应微调）**：可选阶段，利用 MCTS（蒙特卡洛树搜索）对压缩后的结构进行局部优化（如提示词微调），以补偿性能损失。

*   **异构优势**：发现保留少数高能力节点作为“生成者”，而将“验证者”或“聚合者”量化为低成本模型，是最高效的配置。

*   **安全风险**：在安全敏感领域，剪枝和模型替换可能会改变系统的行为模式，存在潜在的可靠性风险。
