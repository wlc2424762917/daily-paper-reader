## 摘要
长程家庭任务需要强大的高层规划和持续的推理能力，而现有的具身智能（Embodied AI）基准测试大多侧重于短程导航或操作，并依赖于固定的任务类别，在很大程度上忽视了这些能力。我们推出了 LongAct，这是一个旨在评估通过自由格式指令指定的长程家庭任务中规划级自主性的基准测试。通过抽象掉特定于具身的底层控制，LongAct 隔离了高层认知能力，如指令理解、依赖管理、记忆维护和自适应规划。我们进一步提出了 HoloMind，这是一个由视觉语言模型（VLM）驱动的智能体，具有基于有向无环图（DAG）的长程分层规划器、用于持久世界建模的多模态空间记忆、用于经验重用的情节记忆（Episodic Memory）以及用于反思性监督的全局评论器（Critic）。使用 GPT-5 和 Qwen3-VL 模型的实验表明，HoloMind 显著提高了长程性能，同时减少了对模型规模的依赖。即使是顶尖模型也仅实现了 59% 的目标完成率和 16% 的全任务成功率，这凸显了 LongAct 的难度以及具身智能体对更强长程规划能力的需求。

## Abstract
Long-horizon household tasks demand robust high-level planning and sustained reasoning capabilities, which are largely overlooked by existing embodied AI benchmarks that emphasize short-horizon navigation or manipulation and rely on fixed task categories. We introduce LongAct, a benchmark designed to evaluate planning-level autonomy in long-horizon household tasks specified through free-form instructions. By abstracting away embodiment-specific low-level control, LongAct isolates high-level cognitive capabilities such as instruction understanding, dependency management, memory maintenance, and adaptive planning. We further propose HoloMind, a VLM-driven agent with a DAG-based long-horizon hierarchical planner, a Multimodal Spatial Memory for persistent world modeling, an Episodic Memory for experience reuse, and a global Critic for reflective supervision. Experiments with GPT-5 and Qwen3-VL models show that HoloMind substantially improves long-horizon performance while reducing reliance on model scale. Even top models achieve only 59% goal completion and 16% full-task success, underscoring the difficulty of LongAct and the need for stronger long-horizon planning in embodied agents.

### 2. 方法论：HoloMind 智能体框架
HoloMind 是一个由视觉语言模型（VLM）驱动的闭环智能体，其核心架构包含四个关键组件：
*   **双层分层规划器**：
    *   **高层规划器（High-level Planner）**：将复杂的长程指令分解为**有向无环图（DAG）**。DAG 能够编码任务间的依赖关系（如必须先打开冰箱才能拿牛奶），并允许独立任务并行考虑以提高效率。
    *   **底层规划器（Low-level Planner）**：将高层目标细化为可执行的子目标序列，并结合记忆信息进行物体接地（Grounding）。
*   **双重记忆系统**：
    *   **多模态空间记忆（Multimodal Spatial Memory）**：构建 3D 语义地图，存储物体的视觉特征（CLIP 嵌入）、空间位置和语义标签，支持在部分观测环境下的物体检索。
    *   **情节记忆（Episodic Memory）**：记录历史动作轨迹，并从中蒸馏出可重用的经验规则（如“切片前必须将物体放在平面上”）。
*   **执行器（Executor）**：调用导航和操作技能库，将规划指令转化为模拟器中的原子动作。
*   **反思性评论器（Critic）**：这是实现闭环的关键。它持续监控执行状态，识别错误并提供三种反馈：**通过（Pass）**、**微调（Refine）**或**重规划（Replan）**。它还能将失败教训转化为经验存入情节记忆。

### 6. 主要结论与发现
*   **架构胜过规模**：在 HoloMind 框架下，较小规模的模型（如 Qwen3-VL-8B）性能提升巨大，甚至能缩小与大模型之间的差距，证明了结构化规划的重要性。

*   **改进率（IR）的有效性**：实验证明纯 VLM 基线的 IR 通常为负，而 HoloMind 能够实现正向的自我改进。

### 7. 优点
*   **基准测试的前瞻性**：LongAct 填补了具身智能在“长程、自由指令、高层认知”评估上的空白。
*   **创新的 IR 指标**：提出了一种动态衡量智能体“越做越好”能力的数学方法，而非仅仅关注最终结果。
*   **模块化设计**：HoloMind 的设计解耦了感知、规划与执行，具有良好的可解释性和 Sim-to-Real 迁移潜力。

### 8. 不足与局限
*   **底层执行瓶颈**：实验显示 46.3% 的失败源于底层执行（尤其是精细操作），这表明高层规划再好，若底层控制不够鲁棒，整体表现依然受限。
*   **被动式规划**：目前的智能体主要在失败后才进行重规划（Reactive），缺乏主动探索环境以消除歧义的主动性（Proactive）。
