## 摘要
随着大模型从对话助手演变为自主智能体，长程决策、工具使用以及真实环境交互带来的挑战日益凸显。现有的智能体基础设施在评估、数据管理和智能体演化方面仍然处于碎片化状态，导致难以系统地发现风险并在持续的闭环中改进模型。在本报告中，我们提出了 Safactory，这是一个用于可信自主智能的可扩展智能体工厂。Safactory 集成了三个紧密耦合的平台：用于轨迹生成的并行仿真平台、用于轨迹存储和经验提取的可信数据平台，以及用于异步强化学习和同策略蒸馏的自主演化平台。据我们所知，Safactory 是首个为下一代可信自主智能提出统一演化流水线的框架。

## Abstract
As large models evolve from conversational assistants into autonomous agents, challenges increasingly arise from long-horizon decision making, tool use, and real environment interaction. Existing agenticinfrastructure remain fragmented across evaluation, data management, and agent evolution, making it difficult to discover risks systematically and improve models in a continuous closed loop. In this report, we present \textbf{Safactory}, a scalable agent factory for trustworthy autonomous intelligence. Safactory integrates three tightly coupled platforms: a \textbf{Parallel Simulation Platform} for trajectory generation, a \textbf{Trustworthy Data Platform} for trajectory storage and experience extraction, and an \textbf{Autonomous Evolution Platform} for asynchronous reinforcement learning and on-policy distillation. As far as we know, Safactory is the first framework to propose a unified evolutionary pipeline for next-generation trustworthy autonomous intelligence.

### 2. 方法论：核心思想与关键技术
Safactory 由三个紧密耦合的平台组成：
*   **并行模拟平台 (PSP)**：
    *   **核心思想**：解决长程任务评估效率低和覆盖不足的问题。
    *   **关键技术**：**回溯沙箱 (Backtracking Sandbox)**。类似于 Git 的版本管理，它能记录 Agent 执行过程中的环境状态快照。当 Agent 触发风险或需要探索不同决策分支时，系统可直接回滚到特定锚点，无需从头开始，极大地节省了 Token 和时间成本。
*   **可信数据平台 (TDP)**：
    *   **核心思想**：将交互轨迹转化为可检索、可审计的资产。
    *   **关键技术**：**DataElf 智能引擎**。采用意图驱动架构，用户通过自然语言描述任务，系统自动编排流水线。包含数据价值评估（DataElf Scorer）、安全审计（23 种检查器）和轨迹经验提取工具。
*   **自主演化平台 (AEP)**：
    *   **核心思想**：支持大规模异步强化学习（RL）和策略优化。
    *   **关键技术**：**Bridge 模块**。实现采样（Rollout）与训练（Train）的解耦，通过版本控制和轨迹陈旧度过滤，确保在异步环境下训练的稳定性。引入**同策略蒸馏 (OPD)**，利用教师模型信号约束策略更新。

### 6. 主要结论与发现
*   **能力与安全的非线性关系**：在桌面环境（OS）中，能力强的模型往往更激进，安全风险更高；而在移动端（Android），能力与安全呈正相关。
*   **具身智能的巨大挑战**：所有模型在具身场景（Embodied Gym）中均表现不佳，处于低成功率、低安全率的“双低”区域，说明多模态感知与规划仍是瓶颈。
*   **经验注入的有效性**：通过 TDP 提取的轻量化经验（Skills）注入 Kimi-K2.5 后，其在 OS 环境下的安全率从 32.17% 大幅提升至 71.08%。
*   **系统效率**：异步 Bridge 架构能有效解决长程 Agent 任务中采样与训练速度不匹配的问题。

### 7. 优点
*   **系统性强**：打破了评估、数据、训练的孤岛，提供了端到端的闭环基础设施。
*   **工程创新**：回溯沙箱（Git-like）和异步训练架构非常契合 Agent 任务长路径、高成本的特点。
*   **国产化适配**：深度集成 DeepLink，为国产算力平台提供了成熟的 Agent 训练方案。
