## 摘要
OpenClaw 生态系统中的基准测试迄今为止仅评估了助手级任务，使得 OpenClaw 的学术级能力在很大程度上未被检验。我们推出了 AcademiClaw，这是一个包含 80 个复杂、长程任务的双语基准测试，这些任务直接源自大学生的真实学术工作流——包括作业、研究项目、竞赛和个人项目——且学生们发现当前的 AI 智能体无法有效解决这些任务。最终的任务集是从 230 个学生提交的候选任务中经过严格的专家评审筛选而出的，涵盖了 25 个以上的专业领域，从奥林匹克级别的数学和语言学问题到 GPU 密集型强化学习和全栈系统调试，其中 16 个任务需要 CUDA GPU 执行。每个任务都在隔离的 Docker 沙箱中执行，并根据结合了六种互补技术的多维评分标准对任务完成情况进行评分，同时通过独立的五类安全审计提供额外的行为分析。对六个前沿模型的实验表明，即使是表现最好的模型也仅达到了 55% 的通过率。进一步的分析揭示了不同任务领域之间明显的的能力边界、模型间迥异的行为策略，以及 Token 消耗与输出质量之间的脱节，提供了超出聚合指标所能揭示的细粒度诊断信号。我们希望 AcademiClaw 及其开源的数据和代码能成为 OpenClaw 社区的有用资源，推动智能体在应对现实世界学术需求的广度上变得更加强大和多能。所有数据和代码均可在 https://github.com/GAIR-NLP/AcademiClaw 获取。

## Abstract
Benchmarks within the OpenClaw ecosystem have thus far evaluated exclusively assistant-level tasks, leaving the academic-level capabilities of OpenClaw largely unexamined. We introduce AcademiClaw, a bilingual benchmark of 80 complex, long-horizon tasks sourced directly from university students' real academic workflows -- homework, research projects, competitions, and personal projects -- that they found current AI agents unable to solve effectively. Curated from 230 student-submitted candidates through rigorous expert review, the final task set spans 25+ professional domains, ranging from olympiad-level mathematics and linguistics problems to GPU-intensive reinforcement learning and full-stack system debugging, with 16 tasks requiring CUDA GPU execution. Each task executes in an isolated Docker sandbox and is scored on task completion by multi-dimensional rubrics combining six complementary techniques, with an independent five-category safety audit providing additional behavioral analysis. Experiments on six frontier models show that even the best achieves only a 55\% pass rate. Further analysis uncovers sharp capability boundaries across task domains, divergent behavioral strategies among models, and a disconnect between token consumption and output quality, providing fine-grained diagnostic signals beyond what aggregate metrics reveal. We hope that AcademiClaw and its open-sourced data and code can serve as a useful resource for the OpenClaw community, driving progress toward agents that are more capable and versatile across the full breadth of real-world academic demands. All data and code are available at https://github.com/GAIR-NLP/AcademiClaw.
